# NeuralVSR: Amortized Verifier-Guided Repair for Diffusion-Generated Placements

> **NeurIPS 2026 submission in progress.** Target: 2-week sprint to paper-ready results.

## What This Is

A **small learned GNN** that corrects constraint violations in chip macro placements generated by diffusion models. Trained on synthetic violation patterns, it **zero-shot generalizes** to real ISPD2005 circuits, reducing placement violations by **37–62%** while being **50× faster** than optimization-based legalization.

**Why it matters**: Diffusion models generate structured outputs but struggle with hard, non-differentiable constraints. Existing solutions require differentiable guidance (can't handle overlap/topology) or expensive test-time optimization (minutes per sample). NeuralVSR amortizes this into a single forward pass.

## Quick Start

### Local development (CPU, tests only)

```bash
pip install -e ".[dev]"
pytest tests/ -v       # 49 tests should pass
```

### GPU experiments (AutoDL recommended)

See `docs/autodl_guide.md` for full setup. Quick version:

```bash
bash scripts/setup_env.sh           # install deps
bash scripts/download_checkpoints.sh  # ChipDiffusion Large+v2
python3 scripts/parse_ispd2005.py --download  # benchmark data

# Run guided baseline + VSR post-processing on ISPD2005
python3 scripts/run_vsr.py \
    --checkpoint checkpoints/large-v2/large-v2.ckpt \
    --task ispd2005 --macro-only --guidance opt \
    --postproc --seed 42
```
